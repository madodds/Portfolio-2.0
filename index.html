Engine Website

The Game Idea Generator randomly generates an idea for a video game project. This was originally written in C# windows forms, which you can find on GitHub.

The idea's attributes are based on the contents of a JSON file on my GitHub account. The attributes chosen are based on randomly generated numbers that are unique to each other. These numbers get placed into the index of 5 arrays: Genres, Perks, Settings, Styles, and Engines.
